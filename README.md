Pre Requerimientos

Instalar React y las siguientes dependencias

- npm i firebase
- npm i react-toastify 
- npm i react-router-dom

Ejecutando la aplicacion

Para ejecutar la aplicacion agregue el siguiente comando en la terminal*

    npm run start

***Luego vaya al: ***

localhost: 3000
